基本原理

  • CSS3画云 
        .cloud
	{
		width:200px;
		height:60px;
		background:#fff;
		border-radius:200px;
		position:relative;
	}
	.cloud:before,.cloud:after
	{
		position:absolute;
		content:'';
		background:#fff;
		width:100px;height:80px;
		border-radius:100px;
		left:10px;
		top:-20px;
		-webkit-transform:rotate(30deg);
	}
	
	.cloud:after
	{
		width:120px;height:120px;
		border-radius:120px;
		left:70px;
		top:-55px;
	}
  • 让云移动

    利用animation与margin属性，让云transform移动

            @-webkit-keyframes moveCloud
		{
			0%{margin-left:1000px;}
			100%{margin-left:-1000px;}
		}

    再为每朵去设置恰当位置与透明度即可

        .x1
	{
		-webkit-animation:moveCloud 15s linear infinite;
		-moz-animation:moveCloud 15s linear infinite;
	}
